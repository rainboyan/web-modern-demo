# Grails Modern Web Demo

A Modern web demo with Sass and esbuild support.

## Grails Version

- Grails **5.0.0**

## Usage

### Build Profile

```
git clone https://github.com/rainboyan/web-modern-demo.git
cd web-modern-demo
npm install
npm sass:build
npm javascript:build
./gradlew bootRun
```
